${ \\mathrm{Example\\enspace 1}} $\n

问题描述\n

已知 $x >0, y > 0$ 且 $2x + 8y - xy = 0$ 求 $x + y$ 的最小值\n

分析与解答\n

通过1的代换来解决问题\n

考虑将 $2x + 8y - xy = 0$ 化简可得：\n

$2x + 8y = xy$ 两边同乘 $\\frac{1}{xy}$ 可得：\n

$ \\frac{2}{y} + \\frac{2}{x} =1 $\n

$ \\because x+y=(x+y)\\cdot 1 $\n

$\\therefore (x+y)=(x+y)(\\frac{8}{x}+\\frac{2}{y} )=\\frac{2x}{y} +\\frac{8y}{x} +10$\n

$\\because x>0,y>0 $\n

$\\therefore \\frac{2x}{y} +\\frac{8y}{x} \\ge2\\sqrt[]{16} $\n

当且仅当 $x = 6, y = 12$ 时 $(\\frac{2x}{y} +\\frac{8y}{x})_{min}=8$ \n

此时 $(x+y)_{min}=18$\n

${ \\mathrm{Example\\enspace 2}} $\n
问题描述\n

已知 $x，y> 0,x+2y+xy-6=0$，解决下列问题：\n

    \n
  1. 求 $xy$ 的最大值\n
  2. $x+2y$的最小值\n
  3. $x+y$ 的最小值\n
  4. $(x+2)^2+(y+1)^2$的最小值\n\n
    分析与解答\n

    $ \\mathrm{{\\Large 第一问} } $\n

    第一问可以使用基本不等式可将 $x+2y\\Rightarrow \\sqrt{2xy}$ 然后进行因式分解：\n

    $\\because x+2y \\ge \\sqrt{2xy},x+2y+xy-6=0$\n

    $\\therefore 2\\sqrt{2xy}+xy-6 \\le 0$\n

    $\\because x,y>0$\n

    $\\therefore (\\sqrt[]{xy} )^2+2\\sqrt[]{2xy} -6\\le 0$\n

    $=(\\sqrt[]{xy} )^2+2\\sqrt[]{2} \\sqrt[]{xy} -6\\le 0$\n

    若将上面的不等式中的 $\\sqrt{xy}$ 看成一个整体，可得一个一元二次不等式，可解得：\n

    $0 < \\sqrt[]{xy} \\le\\sqrt[]{2} $\n

    $0 \\le {xy} \\le 2$\n

    $\\therefore xy_{max}=2$\n

    \n

    $ \\mathrm{{\\Large 第二问} } $\n

    由第一问可得 $xy_{max}=2$\n

    $\\because x+2y=6-xy$\n

    $ \\therefore (x+2y)_{max}=6-xy_{min} = 6-2=4$\n

    \n

    $ \\mathrm{{\\Large 第三问} } $\n

    $\\because x+2y+xy-6=0$\n

    $\\therefore 2y+xy=6-x \\Rightarrow y(2+x)=(6-x) \\Rightarrow y=\\frac{6-x}{2+x} $\n

    $\\therefore x+y=x+\\frac{6-x}{2+x}=x-\\frac{2+x-8}{2+x} =x-1+\\frac{8}{x+2} =x+2+\\frac{8}{x+2} -3$\n

    $\\because x > 0$\n

    $\\therefore x+2+\\frac{8}{x+2} \\ge 2\\sqrt[]{8}$ 当且仅当 $x+2=\\frac{8}{x+2}=\\sqrt{8}$ 时，等号成立，此时 $x=\\sqrt{8}-2$\n

    $\\therefore (x+y)_{min}= (x+2+\\frac{8}{x+2})_{min}-3=4\\sqrt[]{2} -3$\n

    \n

    $ \\mathrm{{\\Large 第四问} } $\n

    *步骤可能可以简化\n

    由第三问可知 $y=\\frac{6-x}{2+x}, x+2+\\frac{8}{x+2} \\ge 2\\sqrt[]{8}$\n

    $\\therefore (x+2)^2+(y+1)=(x+2)^2+(\\frac{6-x}{2+x}+1)=(x+2)^2+(\\frac{8}{2+x})^2=(x+2)^2+\\frac{8^2}{(2+x)^2}$\n

    $\\because \\frac{8^2}{(2+x)^2},(x+2)^2\\ge 0$\n

    $\\therefore (x+2)^2+\\frac{8^2}{(2+x)^2}\\ge 2\\sqrt[]{8^2} =16$ 当且仅当 $ (x+2)^2=\\frac{8^2}{(2+x)^2}=8$ 时成立
    \n此时 $x=\\sqrt{8}-2$\n

    $\\therefore [(x+2)^2+(y+1)^2]_{min}=16$\n

    总结\n

    此题乍看可以使1的代换但深究发现因为有常数的存在而不能使用。此题利用了整体思想\/换元法和消元法。\n

      \n

    • 通过因式分解将多个未知数化简为一个未知数或许能方便运算\n
    • 分子上有未知数时一定要设法删除\n
