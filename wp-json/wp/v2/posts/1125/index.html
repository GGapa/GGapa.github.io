UVA 12661 Funny Car Racing\n

一个标准的最短路，但是在计算距离这一方面需要注意，当我们来当一个赛道上时，会有以下几种情况：\n

    \n
  • 赛道正在开放，且在通过赛道之前赛道不会关闭，时间即为通过赛道的时间\n
  • 赛道正在开放，但是剩余时间不足以通过赛道，时间即为等待时间加上关闭时间加上通过时间。\n
  • 注意连边的时候，若开放时间小于所通过的时间，那么这条边就不用链接了。\n\n

    总结：看到最短路的题目，认真分析新的距离，无需多虑，直接跑最短路就可以，但是对于2023年CSP的一道题目，就需要认真分析性质，然后作答。
    \n最短路的贪心已经是证明过的了，无需再多虑。\n

    UVA 12716 XOR GCD\n

    数论题，我们需要知道：\n

      \n
    • $a \\oplus b = c $ 等价于 $a\\oplus c = b$。\n\n

      那么我们就可以枚举 $a$ 和枚举 $c$，然后算出 $b$，接着验证是否有 $\\gcd(a, b) = c$, 因为 $c$ 时 $a$ 的约数，时间复杂度 $O(n\\log n)$，而计算 $\\gcd$ 的时间是 $O(\\log n)$，总体时间复杂度 $O(n(\\log n)^2)$。\n

      时间复杂度过高，无法通过本题，考虑如何优化。\n

      此时我们暴力打出 $a$、$b$、$c$ 之间的关系，我们可以发现出一个规律 $c = a - b$。\n

      \n

      证明：易得 $a - b\\le a\\oplus b$，且 $a\\oplus b \\ge \\gcd(a, b) = c$，考虑使用反证法，假设 $a -b > c $，可以得到 $c < a - b \\le a\\oplus b$，与 $c = a\\oplus b$ 矛盾，故 $a - b = c$ 得证。\n\n

      时间复杂度 $O(n \\log n)$。\n

      值得注意的是：$\\oplus$ 的运算优先级是低于 $=$ ，在进行判断的时候需要加 $()$。\n

      \n

      但在考场上，我们就不能这么做，我们不需要管证明，我们直接打表看规律！！！\n

      总结：\n

        \n
      • 我们应当牢记：暴力出奇迹，打表出省一！ 谁告诉了我难题一定要推公式才能做呢？\n\n
