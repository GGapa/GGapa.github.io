我去参加了一场考试，有一道题目的输出使我非常的困惑：\n

\n

输出一行一个字符串 $s$ 表示答案。如果这个排列可以通过题目描述的方法构造得到，则 $s=YES$，否则 $s=NO$\n\n

我不会做，便提交了下面的代码并向上天祈求我能通过:\n 

\/*\n天灵灵地灵灵\n随机输出必 AC(accept)\n*\/\n#include <bits\/stdc++.h>\nusing namespace std;\n\nint main() {\n    struct timeb timeSeed;\n    ftime(&timeSeed);\n    srand(timeSeed.time * 1000 + timeSeed.millitm);\/\/用毫秒当作随机数的种子\n    if(rand() % 2) cout << "Yes" << endl;\/\/随机输出\n    else cout << "No" << endl;\n    return 0;\n}\n
刚提交完这道题，考试便提前结束了，这让所有的考生感到非常诧异\n

\n
正当我准备离开考场之际，我背后传来了一声吼叫，我回头看，发现一个很有蟑螂气质的男人指着我。\n

他笑了笑，对我说：“你就是我要找的天选之子。”我听完满嘴问号，他对此并不感到奇怪异，继续说：“你有没有想过为什么你闯了这么多次红灯，却依然健健康康的活到了现在？这件事的概率不会超过百万分之一吧。因为你就是这个故事的「主角」。如果没有你，这个故事就不会继续。这也是为什么你提交了一份随机数代码却依然能在这道没有正解题目上取得满分。”\n

说完他便给我看我的记录，绿油油的非常漂亮，充满着生机，感觉还挺好吃的。\n

我十分好奇那些冥思苦想“正解”的人得知了这个消息之后会怎么想。\n

\n
蟑螂是这个比赛的举办人，他举办了这么多年比赛就是为了找到这个故事的「主角」，而现在他找到了。\n

我和他讨论了很久如何验证我是真正的主角，在排除了跳楼，断头台等一系列众生平等的方案之后，我们决定横穿一条高速公路。我很好奇在高速公路上没有任何可以躲藏的地方我该如何活下来。无论如何，就算我失败了至少能为来去匆忙的人们带来一丝丝喘息的机会。\n

在把钢丝网剪断之后，我们来到了高速公路前。车流飞速而过的气流吸附着我，心中突然有了一种使命感。我感到背后不可抗拒的推力，催促着我迈出第一步。我透过车流看到了高速公路的对面，充满着生机，对比这边的繁华显得格外幽寂。\n

恍惚间我来到了高速公路上，快速的气流使我站不住脚，我摔倒了。我的理性遏制住了我的本能，停在了道路的中间，我想这应该就是我的结局吧，我并不是什么「主角」，就只是一个运气比较好的普通人罢了。我闭上眼睛，等待着最终时刻的到来。\n

以上使我记忆断层前的经历，我不知怎么的就脸朝下趴在了高速公路一旁的草地上。\n

我翻过身来，看到了植物的芬芳。\n

高速公路上的车流停止了，我感到了一丝丝的忏悔，我不知道为什么作者要给我一个好的结局，也许我对他真的非常重要吧。\n

心中总觉得过意不去，于是在路旁用石头刻下了：“对不起，但现在该好好休息一下了。”\n

我走天桥回到了繁华的那一边，仿佛什么都没有发生似的。\n

\n
