1 什么是对数\n

如果 $a^x = N(a>0, a \\neq 1, N>0)$ 则 $x$ 叫做以 $a$ 为底 $N$ 的对数，记做 $x=\\log_aN$ , 其中 $N$ 被称为真数。\n

若对数的底数为 $10$ 则记作 $\\lg N$，若对数的底数为自然底数 $e$，则记作 $\\ln N$ 。\n

2 对数的运算法则\n

2.1 对数的基本运算\n
    \n
  1. $log_a M + log_aN = log_aMN$ \n
  2. $log_aM - log_aN = log_a\\frac{M}{N}$\n
  3. $\\log_ab^x=x\\log_ab$ \n\n

    2.2 换底公式\n

    基本公式：$\\frac{\\log_cb}{\\log_ca} = \\log_ab ,c>0, c\\neq1,a\\neq1$\n

    \n

    证明：已知 $\\log_ab = x \\Leftrightarrow a^x = b$ 则 $\\log_c a^x = \\log_c b$ \n

    由对数的基本运算 $3$ 可知：$x\\log_ca = log_cb$ \n

    故 $x = \\frac{\\log_cb}{\\log_ca} = \\log_ab$\n\n

    扩展1：$\\frac{1}{\\log_ba} = \\log_ab,b\\neq 1$ \n

    \n

    证明：已知 $\\frac{\\log_cb}{\\log_ca} = \\log_ab$\n

    当 $c=b$ 时, $\\frac{1}{\\log_ba} = \\log_ab$\n\n

    扩展2：$\\log_{a^x}b^y = \\frac{y}{x}\\cdot\\log_ab$\n

    \n

    证明：$\\log_{a^x}b^y=y\\log_{a^x}b = y \\cdot \\frac{1}{\\log_ba^x} = y \\cdot \\frac{1}{x\\log_b^a} = \\frac{y}{x}\\cdot\\log_ab$\n\n

    \n

    $\\mathrm{{\\Large Example1} } $\n

    问题描述\n

    求 $\\log_48$ 的值。\n

    分析与解答\n

      \n
    • 方法一：利用换底公式可知 $\\log_48 = \\frac{\\log_28}{\\log_24} = \\frac{3}{2}$\n
    • 方法二：利用换底公式扩展二可直接求出答案 $\\frac{3}{2}$\n\n

      $\\mathrm{{\\Large Example2} } $\n

      题目描述\n

      证明： $\\log_ab \\cdot\\log_bc\\cdot \\log_ca = 1$
      \n分析与解答\n

      此题最大的难点为每个对数的底数不同，为了使他们的底数相同可使用基本换底公式。\n

      原式可化为 $\\log_ab \\cdot \\frac{\\log_ac}{\\log_ab} \\cdot a\\frac{\\log_aa}{\\log_ac} = \\log_aa = 1$ \n

      3.2 对数函数的图像\n

      对于函数 $y=\\log_ax$ ,$a^y = x$：\n

        \n
      • 当 $a > 1 $ 时，$y$ 随着 $x$ 的增大而增大；当 $y=0$ 时， $x = 1$。\n
      • 当 $a < 1$ 时，$y$ 随着 $x$ 的增大而减小；当 $y=0$ 时， $x = 1$。\n\n

        若两个对数函数的底数互为倒数，则两个函数关于 $x$ 轴，对称。\n

        函数的底数越大，函数的增长速度越慢。\n

        $\\mathrm{{\\Large Example1} } $\n

        问题描述\n

        已知 $a=2^{1.1},b=\\log_23,c=3^{\\log_3\\frac{3}{2}}$ ,求 $a,b,c$ 的大小关系。\n

        分析与解答\n

        此题的关键在于估算。\n

        $a = 2^{1.1} \\approx 2^1 \\approx 2$\n

        $b = \\log_23 \\Leftrightarrow 2^b = 3 \\Rightarrow b < 2 < a$\n

        $c =3^{ \\log_3\\frac{3}{2} } = \\frac{3}{2} = \\log_22^\\frac{3}{2} = \\log_2\\sqrt{8} < \\log_23$ \n
